ESP 8266 WI-FI Microcontroller

Boolean Control

Click the button to toggle a control on or off, the microcontroller will grab the states from the database shortly and adjust accordingly

For Pulse Width Modulation: select the PWM checkbox, set the slider to the desired value and then click the button for the desired pin

0

Text Control

Input one of the predetermined command words, then click 'Send' to send the instruction immediately. it will have a status of 'HOLD' and after the microcontroller has received the command the status in the database will change to 'SENT'

Output Display

The default here simply shows the current state of the database, this will certainly need customized for a particular use though.

The access log will show the timestamp of when this page was last (re)loaded, or when the microcontroller last retrieved the instruction set from the database.
