Description

This php script receives data from the microcontroller and adds it to the database.

This script will require two parameters passed in with the URL. One to identify which sensor (ID, number, name) and one holding the value from the sensor.

See the pinout diagram on the Home Page

Parameters

pinID
  • The value printed in silk-screen on the PCB of the microcontroller
pinNumber
  • When setting up the database I began with D0 is numbered 0 and progressing clockwise around to A0 is numbered 29.
  • These are the identifier used most often in the JavaScript/AJAX
sensor
  • The labeling for the sensor connected to a particular pin, obviously this is unique to each project and 'nothing' in the starter files here
pinName
  • The names used in the bootloader for the ESP8266-12E NodeMCU
  • These are the GPIO, ADC, CH_PD, etc. listed on the pinout diagram shown on the Home Page
status
  • The value the sensor will be sending to the database

Returns

String
  • "success"
  • "failure"
