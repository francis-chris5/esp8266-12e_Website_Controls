Description

This php script is used to remove all text-based commands from the database with a status of 'SENT' which indicates the microcontroller has already received the instruction.

Since the tx_command will send only the oldest text-based command entered it is critical to schedule a regular run of this script to clear out old data from that table and future transmissions.

The suggested order for processing text commands in each pass of the loop in the microcontroller, which are sent one at a time is: removeCommands, tx_command, and then commandReceived. See the example page for more details.
