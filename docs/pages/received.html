Description

This php script changes the status in the database for commands that have already been processed by the microcontroller from 'HOLD' to 'SENT' to indicate it is ready to be deleted.

The suggested order for processing text commands in each pass of the loop in the microcontroller, which are sent one at a time is: removeCommands, tx_command, and then commandReceived. See the example page for more details.

Parameters

comID
  • The id field from the database which belongs to the command to relabel with a status of 'SENT'

Returns

void
