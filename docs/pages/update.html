This php script is used to echo the data to the web page user interface to display whatever the user needs.

The default in the starter files is simply an echo of all data currently held in the database with html table tags added in appropriately. The html code echoed back will most certainly need customized for a particular project so that it looks better than a bunch of tables filled with raw data.
