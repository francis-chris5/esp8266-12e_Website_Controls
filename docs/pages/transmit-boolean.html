Description

The transmit boolean script is used to transfer state values of each control as it is stored in the database. This can be used for the obvious purpose of updating the state of the microcontroller, as well as updating the user interface aspect of the web page.

Paramaters

mode
The mode value passed in with the URL determines how the data is to be transmitted
    • x
    • returns the data in xml format
    • the default value if no mode is set in the URL
    • j
    • returns the data in json format
    • c
    • returns the data in csv format
    • NOTE: it is not a .csv file, just that format with linebreaks to easily create a csv file with headers or split into an associative array
    • k
    • returns the data in a custom format
    • the default is a comma delimited list of status values given in the order of pinNumber as listed in the database
    • Personally I modify this if the data needs to be secured, if an outsider doesn't know the format the data is being transferred in there is one more step to decrpyting it

Returns

String
formatted as determined by the mode paramater
